# Отдельная сцена с меню (3 кнопки)

На втором скрине меню без кнопок — ниже нужно добавить три кнопки в тёмно-синюю панель: **PLAY**, **OPTIONS**, **QUIT**.

## 1. Новая сцена меню

1. В **Assets** создайте сцену: правый клик по папке `assets/Scene` → **New** → **Scene**. Назовите, например, `MenuScene`.
2. Откройте сцену `MenuScene`: в ней должен быть узел **Canvas** и ваше оформление:
   - сверху — розовый «пузырь» с текстом **BLAST GAME!**;
   - ниже — тёмно-синяя скруглённая панель (пустая на втором скрине).

## 2. Три кнопки внутри панели

Внутри тёмно-синей панели создайте **3 дочерних узла** — кнопки:

1. **Play**  
   - Узел с текстом "PLAY" и иконкой бомбы (как на первом скрине).  
   - Добавьте компонент **Button** (Add Component → UI Component → Button), если его ещё нет.

2. **Options**  
   - Узел с текстом "OPTIONS" и иконкой рюкзака.  
   - Компонент **Button**.

3. **Quit**  
   - Узел с текстом "QUIT" и иконкой стрелок.  
   - Компонент **Button**.

Расположите кнопки вертикально с отступами (Layout по желанию: Vertical Layout или вручную).

## 3. Скрипт MainMenu

1. Выберите **Canvas** (или корневой узел сцены меню).
2. **Add Component** → найдите скрипт **MainMenu** и добавьте его.
3. В инспекторе у **MainMenu** заполните:
   - **Play Btn** — перетащите узел кнопки PLAY;
   - **Options Btn** — узел кнопки OPTIONS;
   - **Quit Btn** — узел кнопки QUIT;
   - **Game Scene Name** — оставьте `GameScene` (имя сцены с игрой без расширения).

## 4. Поведение кнопок

- **PLAY** — загружает сцену с игрой (`cc.director.loadScene('GameScene')`). Имя сцены задаётся в поле **Game Scene Name**.
- **OPTIONS** — пока заглушка (в консоли "Options"); позже можно повесить панель настроек или отдельную сцену.
- **QUIT** — в браузере попытка закрыть окно (если доступно), в нативном приложении — выход из игры.

## 5. Старт с меню

В **Project** → **Project Settings** → **Launch Scene** выберите сцену **MenuScene**, чтобы при запуске игры открывалось меню, а не игровая сцена.

---

Итог: у вас отдельная сцена меню с тремя кнопками в синей панели; скрипт **MainMenu** обрабатывает нажатия и по PLAY переходит на сцену с игрой.
